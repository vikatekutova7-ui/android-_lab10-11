# Лабораторная работа №10-11. Создание приложения «Список студентов» с использованием Material Design
### **Цель работы:** 
Научиться применять принципы Material Design в Android приложениях с использованием Jetpack Compose. В результате лабораторной работы будет создано приложение «Список студентов», которое демонстрирует использование цветовых схем, типографики, форм и анимаций Material Design.


**Material Design** — это система дизайна, разработанная Google для создания
качественных цифровых продуктов для Android, веб и других платформ. Она
предоставляет рекомендации по созданию читабельного, привлекательного и
согласованного пользовательского интерфейса.


## Функциональность
- Отображение списка студентов с фотографиями, именами и возрастом
- Раскрывающиеся карточки с подробной информацией о каждом студенте
- Плавная анимация при раскрытии и сворачивании карточек
- Поддержка светлой и тёмной тем
- Верхняя панель приложения с логотипом и названием
- Круглые аватарки студентов с правильным кадрированием изображений
- Индивидуальные скругления углов для карточек

## Технологии
- **UI Toolkit:** Jetpack Compose
- **Дизайн-система:** Material Design 3
- **Анимации:** Spring Animation
- **Шрифты:** Abril Fatface, Montserrat (Google Fonts)
- **Иконки:** Material Icons Extended
